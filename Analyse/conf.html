Conférences

Key Performance

Key Performance est une entreprise spécialisée dans les performances de sites Web.

Le CEO de l’entreprise Michel Kalis nous a expliqué l’importance du ressentiment de l’utilisateur, il nous a parlé des différentes métriques pour mesurer la performance et des outils permettant de les calculer.

La vitesse d’un site influe énormément sur les revenus que celui-ci génère ainsi que sur le référencement de google, il est donc très important de ne pas sous-estimer cette étape lors de la conception d’un site Web.

Cette conférence fut tellement intéressante que j’ai décidé d’aller effectuer mon stage chez Key Performance.
